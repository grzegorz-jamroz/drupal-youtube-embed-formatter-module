# Configuration

1.  After enabling the module, navigate to the configuration page at `Manage > Structure > Content types` (`/admin/structure/types`) and select "Manage fields" for a desired content type (e.g., Article).

    ![Configuration](assets/configuration-1.png "Configuration")
    ![Configuration](assets/configuration-2.png "Configuration")

3. Select "Create a new field" and choose "Plain text" as the field type. Set the field label (e.g., "YouTube Video") and save.

   ![Configuration](assets/configuration-3.png "Configuration")
   ![Configuration](assets/configuration-4.png "Configuration")
   ![Configuration](assets/configuration-5.png "Configuration")

3.  Select "Manage display" tab for the same content type.
4.  Locate the newly created field and select "Responsive YouTube Embed" from the "Format" dropdown menu.

    ![Configuration](assets/configuration-6.png "Configuration")

5.  Click the gear icon next to the format dropdown to configure the formatter settings:
    -   **Show Player Controls:** Choose whether to display the YouTube player controls.
    -   **Enable Privacy-Enhanced Mode:** Choose whether to use YouTube's privacy-enhanced mode (no cookies).

    ![Configuration](assets/configuration-7.png "Configuration")

6. Save the settings.
